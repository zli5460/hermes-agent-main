# Phoenix Architecture V4.7

## 板块结构
1. Core — 核心配置/状态/任务管理
2. Router — 双维度智能路由（关键词+LLM理解）
3. Executor — 任务分解/预审批/微压缩/深度压缩
4. Memory — 三层记忆系统（短期+事实+长期）
5. Self-heal — 抗体库/错误处理/进化引擎/故障剧本
6. Integration — Gateway API/路由钩子
7. Security — 审批系统/权限管理/Token追踪
8. Adapt — Hermes自动适配
9. Sandbox — Docker沙箱执行
10. Workflow — 持久化工作流
11. GitHub — GitHub集成
