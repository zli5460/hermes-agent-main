## 1. 这东西是干什么的？

不死鸟是挂在 **Hermes Agent** 上的 **插件 + 配置**：让你在 **不换聊天习惯** 的前提下：

- **平时**用你自己配的「默认模型」聊天；  
- **需要更强能力时**，手动输入 **`/深度`、`/大神`、`/真神`**，系统在你要花钱前先 **确认**；  
- **模型挂了**时，按预设自动走 **兜底**，而不是整条对话崩掉。

---

## 2. 和 Hermes 是什么关系？

| 部分 | 说明 |
|------|------|
| **Hermes** | 本体：终端命令 `hermes`、Gateway、工具、各聊天通道。 |
| **不死鸟** | 在 Hermes 之上增加：**路由口令**、**确认流**、**档内 fallback**、以及 **`~/.hermes/phoenix`（`$PHOENIX_HOME`）目录里的记忆/自愈等模块**。 |

你不死鸟 **不会替代** Hermes；聊天入口仍然是 Hermes，只是消息会先经过插件策略判断。

---

## 3. 日常怎么用（不花钱升档）

直接说话即可，例如：

```text
你好
帮我总结下面这段话
这张图里有什么
```

不要输入 **`/深度`** 这类前缀时，**默认不会**因为「话题很深」就自动切到最贵的模型。

---

## 4. 高档怎么用（三条口令）

必须在消息里 **显式** 带前缀（与后续正文之间建议空格）：

| 口令 | 典型用途 |
|------|----------|
| **`/深度`** | 深度分析、复杂排查 |
| **`/大神`** | 架构级、大体量方案 |
| **`/真神`** | 顶配推演（以你安装时绑定的模型为准） |

示例：

```text
/深度 帮我把这个仓库的模块依赖梳理一遍
/大神 我要做一个多端同步的方案，给大纲
/真神 帮我把商业模式和风险点列全
```

## 5. 确认框三件套

看到确认提示后，一般用文字回复其一：

| 回复 | 含义 |
|------|------|
| **确认** | 同意走当前高档；失败则按配置尝试兜底 |
| **降级** | 不用高档，用默认模型继续做 |
| **取消** | 本条高档任务不执行 |

（具体交互文案随 Hermes / 通道略有差异，含义不变。）

## 6. 各通道是否一样（CLI / TUI / Telegram / 飞书 / 微信）

**路由规则同一套**：都是用户一句消息进 Hermes，插件层统一拦 `/深度` 等口令。  
**展示形式**可能不同：有的通道弹窗明显，有的通道是纯文字确认——以你当前客户端为准。

## 7. 装完后想改模型或 Key

1. **改 Key**：编辑 **`~/.hermes/.env`** 里 `PHOENIX_*` 变量（勿泄露给他人）。  
2. **改模型名 / 网关地址**：编辑 **`~/.hermes/phoenix/config.json`** 中 `phoenix_open`，或 **重新运行安装脚本**（会先备份）。  
3. 修改后执行：**`hermes gateway restart`**（或 `start`）。

默认 Hermes 全局模型见 **`~/.hermes/config.yaml`** 顶层 `model` 段；若与「日常模型」不一致，以你们运维规范为准（必要时改成一致）。

---

## 8. 网关 Gateway

不死鸟改配置后，需要 Gateway 进程加载新配置。安装脚本一般会尝试重启；手动时可：

```bash
hermes gateway restart
# 或
hermes gateway start
hermes gateway status
```

---

## 9. 出问题先干什么

按顺序：

1. `hermes gateway status`  
2. `python3 ~/.hermes/phoenix/doctor.py --verify --home ~/.hermes --phoenix-home ~/.hermes/phoenix --hermes-agent-dir <你的 hermes-agent 路径>`  
3. 看 **`~/.hermes/phoenix/data/`** 或插件日志里是否有明显报错（若有）  
4. 仍不行：备份 `~/.hermes` 后重装 Hermes 或不死鸟安装包（见 `RELEASE_交付流程与验收.md`）

---

## 10. 不要做的事

- 不要在 **聊天窗口** 里贴正式 API Key。  
- 不要把 **`hermes` 命令** 改成指向不死鸟自己的 CLI（会破坏 Hermes）。  
- 不要以为「我说得很专业」系统就会自动升档——**只有三条口令**才会进入高档流程。

## 11. 常见问题

**Q：为什么「深度学习」四个字不会进深度模式？**  
A：V8 故意只做 **`/深度` 前缀**，避免自然语言误触贵模型。

**Q：记忆存在哪？**  
A：见 **`不死鸟_Phoenix_V8_技术细则与路径原理.md`** 记忆章节。

**Q：Hermes 升级后不死鸟要做什么？**  
A：见 **`HERMES_融合与随版本升级.md`**。
